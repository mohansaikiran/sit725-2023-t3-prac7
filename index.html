SIT 725 Week 5 Workshop on MVC Architecture

This is a sample application to demonstrate the Model View Controller Architecture design. The Model represents the application's data and business logic. It encapsulates the data and provides methods for manipulating and accessing it. The View is responsible for displaying the application's user interface. It presents the data from the Model to the user and captures user input. The Controller acts as an intermediary between the Model and the View.
